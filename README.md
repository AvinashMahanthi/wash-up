# WashUp
An Application for booking car-wash

### Install Dependencies (Client)

```
cd frontend/
npm i
```

### Install Dependencies (Server)

```
cd server/
npm i
```

### Run Frontend in LocalHost

```
cd frontend/
npm start
```

### Run Backend in LocalHost

```
cd server/
nodemon app
```
